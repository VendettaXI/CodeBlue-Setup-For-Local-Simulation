# CodeBlue Dating App

A polished, componentized React + Vite app themed for healthcare professionals.

## Quick start

- Install dependencies: npm install
- Run the dev server: npm run dev
- Build for prod: npm run build
- Preview build: npm run preview

## Project status and docs

Docs have moved to the `docs/` folder to keep the root clean.

## Tech stack

- React 19, Vite, Tailwind CSS, Lucide Icons

## Folder structure

- src/CodeBlueDating.jsx – main app shell and embedded styles
- src/components/** – reusable UI components and tabs

## Notes

- The app uses white card backgrounds with filled buttons and a "Floating Island" bottom nav.
- Colors follow an NHS-inspired blue–teal palette with contextual tab accents.
- Typography uses a global system font stack with refined heading letter-spacing for a premium feel.
